ИНСТРУКЦИЯ ПОЛЬЗОВАТЕЛЯ ДЛЯ ПРОГРАММЫ chip_ore_brightout

УСТАНОВКА И НАСТРОЙКА ОКРУЖЕНИЯ

1. Убедитесь, что у вас установлен Python 3.x на вашем компьютере.
   - Рекомендуется использовать Python 3.8 или выше
   - Скачать можно с официального сайта python.org

2. Скопируйте все файлы из этой папки (for_sharing) в новую директорию на вашем компьютере.

3. Активируйте виртуальное окружение:
   - Откройте командную строку (cmd) или терминал
   - Перейдите в папку, куда вы скопировали файлы
   - Выполните команду для активации виртуального окружения:
   
   На Windows:
   cd stereo_venv\Scripts
   activate.bat
   cd ..\..
   
   На Linux/Mac:
   source stereo_venv/bin/activate

4. После активации виртуального окружения запустите программу:
   python chip_ore_brightout.py

5. Если возникнут ошибки при запуске, проверьте зависимости:
   pip list
   - В списке должны присутствовать такие библиотеки как opencv-python, numpy и другие
   
6. Для деактивации виртуального окружения после работы выполните:
   deactivate

Дополнительные зависимости можно установить командой:
pip install -r requirements.txt
(если файл requirements.txt будет предоставлен разработчиком)

Примечание: Виртуальное окружение уже содержит основные зависимости, необходимые для работы программы.

ИСПОЛЬЗОВАНИЕ ПРОГРАММЫ

1. Подготовка изображения:
   - Поместите изображение (в идеале разрешение должно быть 2048х2048, СЕМ дает чуть больша за счет плашки с информацией) с чипом в ту же папку, где находятся скрипты
   - Измените путь к изображению в строке 275 файла chip_ore_brightout.py на путь к вашему изображению

2. Настройка параметров:
   - Все основные параметры находятся в файле detection_config.json
   - Вы можете изменять параметры без редактирования основного кода (однако я сама редачу из прям в коде)

Приложила полученное фото с СЕМ, обрезанный вариант в виде jpg и финальную обработку
Основные параметры в detection_config.json:

- brightness_threshold (сейчас: 56): 
  Нижний порог яркости для определения наличия частицы в лунке.
  Если средняя яркость лунки ниже этого значения, лунка считается пустой.

- brightness_upper_threshold (сейчас: 160):
  Верхний порог яркости. Если яркость выше этого значения, 
  это может быть блик или артефакт, а не частица.

- uniformity_threshold (сейчас: 41):
  Порог однородности (стандартное отклонение). 
  Если стандартное отклонение яркости выше этого значения,
  лунка может содержать мусор/дефекты вместо нормальной частицы.

Параметры для HoughCircles (обнаружение лунок, лучше подстраиваться под эти параметры при работе на СЕМ и делать фотографии при фиксированном увеличении):

- min_dist (по умолчанию: 8):
  Минимальное расстояние между центрами обнаруженных лунок

- param1 (по умолчанию: 50):
  Верхний порог для детектора границ Canny

- param2 (по умолчанию: 28):
  Порог для центра круга (чем меньше значение, тем больше ложных срабатываний)

- min_radius (по умолчанию: 7):
  Минимальный радиус лунки в пикселях

- max_radius (по умолчанию: 12):
  Максимальный радиус лунки в пикселях

3. Альтернативный способ использования - GUI версия:
   - Запустите: python chip_ore_brightout_simple_gui.py
   - В графическом интерфейсе доступны ползунки для настройки всех параметров в реальном времени

4. Результаты работы:
   - Программа выводит статистику в консоль
   - Создает визуализацию с отмеченными лунками (сохраняется как wells_analysis_result.jpg)
   - Разные цвета обозначают разные типы: зеленый - есть частица, красный - пусто, оранжевый - мусор, пурпурный - вне лунки
